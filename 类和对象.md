#类和对象
##类
Java中把具有相同性质、方法的一组对象的集合作为“类”。定义方法：
```java
[修饰] class ClassName {
}
```
类名的首字母应大写。
在类中可以定义属性、方法和构造函数。
```java
int attr1;
String attr2;
......
public static void method(){
    //ToDo
}
......
className (String name){
    //ToDo
}
```
构造函数不能定义返回值
构造函数具体语法说明：
1.修饰符：可省略或为public,private,protected之一
2.构造函数名与类名相同
3.参数语法与方法中参数的语法一致
##对象
定义类后，可以生成其对象，并定义属性和方法。
